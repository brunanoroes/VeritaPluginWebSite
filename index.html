O que o VeritaPlugin detecta

Seis categorias cobertas pelo modelo treinado

💰

Golpes Financeiros

Promessas falsas de dinheiro fácil, prêmios ou rendimentos absurdos.

📰

Desinformação Digital

Notícias falsas criadas com fins maliciosos para enganar e manipular.

🛒

Lojas Virtuais Falsas

E-commerces fraudulentos que cobram sem entregar produtos.

🎣

Phishing

Coleta indevida de dados pessoais, senhas e informações bancárias.

💔

Golpes de Relacionamento

Manipulação emocional para obter dinheiro ou dados da vítima.

✅

Seguro

Publicação sem indícios de golpe detectados pelo modelo.

Instalação

Três passos e o plugin estará pronto no seu navegador

1

Baixe o projeto

Acesse o repositório no GitHub e baixe o projeto como ZIP ou clone pelo terminal. Extraia em uma pasta que você vá lembrar.

Abrir no GitHub
2

Baixe o modelo BERTimbau ~416 MB

O modelo de IA não está incluso no repositório por ser muito grande. Baixe o arquivo model.safetensors e cole dentro da pasta modelo_bert/ do projeto. 

VeritaPlugin/
└── modelo_bert/
    ├── config.json
    ├── model.safetensors   ← colocar aqui
    ├── tokenizer.json
    └── tokenizer_config.json
Baixar modelo no Hugging Face
3

Carregue a extensão no seu navegador

Todos os navegadores baseados em Chromium suportam extensões não empacotadas. Escolha o seu:

  1. Abra o Chrome e acesse chrome://extensions/
  2. Ative o Modo do desenvolvedor (canto superior direito)
  3. Clique em Carregar sem compactação
  4. Selecione a pasta verita-plugin/ dentro do projeto baixado
  1. Abra o Edge e acesse edge://extensions/
  2. Ative o Modo do desenvolvedor (menu lateral esquerdo)
  3. Clique em Carregar sem compactação
  4. Selecione a pasta verita-plugin/ dentro do projeto baixado
  1. Abra o Brave e acesse brave://extensions/
  2. Ative o Modo do desenvolvedor (canto superior direito)
  3. Clique em Carregar sem compactação
  4. Selecione a pasta verita-plugin/ dentro do projeto baixado
  1. Abra o Opera e acesse opera://extensions/
  2. Ative o Modo do desenvolvedor (canto superior direito)
  3. Clique em Carregar sem compactação
  4. Selecione a pasta verita-plugin/ dentro do projeto baixado
✨ Após carregar a extensão, um assistente de configuração abrirá automaticamente e te guiará pelo restante da instalação.

VeritaPlugin

Extensão do Chrome para detecção de golpes digitais no Facebook, desenvolvida como Trabalho de Conclusão de Curso (TCC). Combina um modelo BERTimbau treinado para classificação de golpes com um pipeline RAG jurídico para fornecer análises fundamentadas em lei.
