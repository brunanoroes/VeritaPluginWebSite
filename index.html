O que o VeritaPlugin detecta

Seis categorias cobertas pelo modelo treinado

💰

Golpes Financeiros

Promessas falsas de dinheiro fácil, prêmios ou rendimentos absurdos.

📰

Desinformação Digital

Notícias falsas criadas com fins maliciosos para enganar e manipular.

🛒

Lojas Virtuais Falsas

E-commerces fraudulentos que cobram sem entregar produtos.

🎣

Phishing

Coleta indevida de dados pessoais, senhas e informações bancárias.

💔

Golpes de Relacionamento

Manipulação emocional para obter dinheiro ou dados da vítima.

✅

Seguro

Publicação sem indícios de golpe detectados pelo modelo.

Instalação

Três passos e o plugin estará pronto no seu navegador

1

Baixe o projeto

Primeiro você precisa baixar os arquivos do plugin para o seu computador. Clique no botão abaixo, aguarde o download terminar e depois extraia a pasta ZIP normalmente (botão direito → "Extrair aqui"). Guarde bem onde você salvou — você vai precisar dessa pasta logo mais.

Recomendado Usuário iniciante
📦

Download ZIP

Baixa o projeto pronto como um arquivo compactado. Só extrair e usar.

Baixar ZIP
ou
Avançado Usuário técnico
💻

Clonar com Git

Ideal se você já tem o Git instalado e prefere usar o terminal.

git clone https://github.com/brunanoroes/VeritaPlugin.git
2

Baixe o modelo BERTimbau ~416 MB

O plugin usa uma inteligência artificial para detectar golpes. O arquivo dela é grande demais para vir junto com o projeto, então você precisa baixá-lo separado. Após baixar, cole o arquivo model.safetensors dentro da pasta modelo_bert que está dentro da pasta que você baixou no Passo 1. 

VeritaPlugin/
└── modelo_bert/
    ├── config.json
    ├── model.safetensors   ← colocar aqui
    ├── tokenizer.json
    └── tokenizer_config.json
Baixar modelo no Hugging Face
3

Carregue a extensão no seu navegador

Agora é hora de instalar o plugin no seu navegador. Não precisa comprar nem pagar nada — é só carregar a pasta que você baixou. Selecione o navegador que você usa:

  1. Abra o Chrome e na barra de endereços (onde você digita sites) cole o endereço abaixo e pressione Enter: chrome://extensions/
  2. No canto superior direito da tela que abriu, ative a chave "Modo do desenvolvedor"
  3. Vai aparecer um botão "Carregar sem compactação" — clique nele
  4. Uma janela de arquivos vai abrir. Navegue até a pasta que você baixou no Passo 1 e selecione a pasta chamada verita-plugin

  4. Uma janela de arquivos vai abrir. Navegue até a pasta que você baixou no Passo 1 e selecione a pasta chamada verita-plugin

VeritaPlugin

Extensão do Chrome para detecção de golpes digitais no Facebook, desenvolvida como Trabalho de Conclusão de Curso (TCC). Combina um modelo BERTimbau treinado para classificação de golpes com um pipeline RAG jurídico para fornecer análises fundamentadas em lei.
